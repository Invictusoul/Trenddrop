# TrendDrop - Self-Selling Trending Digital Store

Auto-adapts products every 1-2 days based on trends. Niche selector included.

## Quick Start
1. `pip install -r requirements.txt`
2. Copy `.env.example` to `.env` and add Stripe keys
3. `cd trenddrop && PYTHONPATH=. uvicorn app.store:app --reload --port 8000`

Visit http://localhost:8000

## Deploy (Free Options)
- **Vercel / Render / Railway**: Push to Git, set env vars.
- For cron updates: Add a scheduler or use GitHub Actions.
